# REST API

Prerequisites:
1. You need to install the LTS version of NodeJS (https://nodejs.org)
2. Make sure that node is running. After entering the following command, the version of installed Node.js should be displayed:
```
node --version
```

Launching the project:
1. open the console
2. navigate to the `nestjs` folder:

```
cd nestjs
```

3. Install NPM packages:
```
npm install
```

4. Launch the project by:
```
npm run start
```
